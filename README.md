#  CineAI – AI Cinematic Multi-Focus Camera

CineAI is an AI-powered cinematic camera system that automatically detects, tracks, and focuses on multiple objects in real time. It provides DSLR-like cinematic effects using deep learning and computer vision.

##  Live Demo

👉 https://cineai-2a2b9.web.app

##  Features

✔ Multi-object detection
✔ Dynamic cinematic focus
✔ Auto object tracking
✔ Background blur (bokeh effect)
✔ Tap-to-focus
✔ Multi-focus (more than one person)
✔ Screen recording and download
✔ Android mobile support
✔ Real-time AI processing


##  Technologies Used

* TensorFlow.js
* COCO-SSD object detection
* React.js
* Firebase Hosting
* Android WebView
* HTML5 Canvas
* MediaRecorder API


## Android App

The CineAI Android app allows users to use cinematic AI camera features directly on mobile devices.
